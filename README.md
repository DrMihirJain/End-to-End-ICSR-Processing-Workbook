# End-to-End ICSR Processing Workbook
### A Simulated Pharmacovigilance Literature Case Processing Portfolio

![Case Analysis Summary](Asset/Pharmacovigilance_Case_Analysis_Summary.png)

---

## Project Overview

This repository demonstrates an end-to-end simulation of Individual Case Safety Report (ICSR) processing based on published literature case reports. The objective is to replicate the workflow performed by entry-level Pharmacovigilance Associates in literature surveillance and case processing teams.

Each published case report has been processed through a structured workflow including data extraction, medical assessment, causality assessment, seriousness evaluation, and completion of a CIOMS I form.

> **Disclaimer:** This repository is created solely for educational and portfolio purposes. It does not contain confidential patient data or proprietary safety database information. All cases are derived from publicly available published literature.

---

## Objectives

- Simulate real-world pharmacovigilance literature case processing
- Demonstrate understanding of ICSR workflows
- Apply structured medical review and safety assessment
- Practice regulatory documentation standards
- Showcase documentation and analytical skills relevant to Pharmacovigilance roles

# Repository Contents

## 1. Project Methodology

Contains the methodology report describing:

- Literature selection criteria

---

## 2. Case Tracker

An Excel workbook used to monitor all processed cases throughout the project.

Typical fields include:

- Case ID
- Drug
- Adverse Event
- Source
- Publication Year
- Seriousness
- Outcome
- Processing Status
- Completion Status

---

## 3. Processed Cases

Each processed case is maintained in an individual folder containing:

### Published Case Report

Original peer-reviewed literature article used as the source.

### Processing Report

Structured medical review including:

- Patient demographics
- Medical history
- Drug exposure
- Adverse event chronology
- MedDRA Coding
- Clinical assessment
- Seriousness determination
- Expectedness assessment
- Causality assessment
- Medical narrative

### Completed CIOMS I Form

A completed CIOMS I form prepared using information extracted from the published literature.
